# Quick 260719-26c: Add Fallow config + package.json scripts/devDependency + fallow CI job Summary

Fallow is now a first-class, project-owned, CI-gated tool: a committed `.fallowrc.jsonc` declares the known import-graph false-positives so `fallow dead-code --fail-on-issues` is clean over the whole repo, `fallow@~3.6.0` is a pinned devDependency with `fallow`/`fallow:ci` scripts, and a base-independent `fallow` CI job gates every push and PR.

## What was done

### Task 1 - fallow devDependency + scripts (commit cd3d05b)
- Added `"fallow": "~3.6.0"` to root `devDependencies` (alphabetical, before `nx`).
- Added scripts: `"fallow": "fallow dead-code"` (local human-readable report, exit 0 even with findings) and `"fallow:ci": "fallow dead-code --fail-on-issues"` (the CI gate, exit 1 on any finding).
