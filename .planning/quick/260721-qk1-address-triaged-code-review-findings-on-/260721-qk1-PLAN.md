## Commit 1 - fix(retention): clamp sub-1-day CACHE_MIRROR_MAX_AGE_DAYS to 1

- `resolveMaxAgeDays` returns `Math.min(Math.floor(raw), CEILING)`, which yields 0 for
  any value in (0,1). cleanup.ts then sets `cutoff = Date.now()` and prunes the entire
  mirror, violating the retention-locked rule. Change to
  `Math.max(1, Math.min(Math.floor(raw), MAX_AGE_CEILING_DAYS))`; update the doc comment.
- Add spec cases pinning `'0.5'` and `'0.99'` -> 1.
- verify: `nx test github-cache`
- done: retention.spec passes incl. the new sub-1 cases.
